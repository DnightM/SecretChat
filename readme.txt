[단축키]
1. 키보드 우측 키패드 [+] 키
    화면 숨기기.
    다시 한번 누르면 숨겼던 창이 활성화 됨.

2. 키보드 우측 키패드 [-] 키
    폰트 설정 옵션

3. 키보드 우측 키패드 [*] 키
    스크롤을 맨 아래로 설정.
    대화창 스크롤이 고정되어 자동 업데이트가 필요한 경우 사용.

4. [F11] 키
    강제 종료

5. 좌측 하단 [CTRL] 키
    화면을 움직이는 단축키
    누르고 있는 동안 화면이 마우스를 따라 움직임.

6. 화면 크기 조절 [SHIFT] + 키보드 우측 키패드 [+],[-] 키
    SHIFT를 누른 상태에서 +를 누르면 화면이 아래로 확장됨.
    SHIFT를 누른 상태에서 -를 누르면 화면이 축소됨.
